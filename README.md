# SmartBusPass 🚍

A cloud-based bus pass booking system demonstrating DevSecOps practices.

## 🔗 Live Demo
https://smartbuspass-wowa.onrender.com

## 📌 Features
- User authentication (JWT)
- Route search and booking
- Digital pass management
- Secure API with rate limiting

## 🛠 Tech Stack
- Frontend: HTML, React (CDN)
- Backend: Node.js, Express
- Database: SQLite
- Deployment: Render

## 🔐 DevSecOps Practices
- JWT-based authentication
- Password hashing (bcrypt)
- Rate limiting (API protection)
- Environment variables for secrets
- Cloud deployment

## 🚀 How to Run Locally
1. Clone repo
2. Run `npm install`
3. Run `node server.js`
4. Open browser at `http://localhost:3001`

## 📁 Project Structure
- routes/ → API routes
- middleware/ → authentication logic
- db/ → database setup
- public/ → frontend UI
